# 🎮 بازی دوز (Tic-Tac-Toe) با هوش مصنوعی قابل تنظیم

یک بازی دوز گرافیکی ساخته شده با **Python** و کتابخانه **Tkinter**.  
در این بازی می‌توانید با یک هوش مصنوعی رقابت کنید که سطح هوشمندی آن بین ۰٪ (کاملاً تصادفی) تا ۱۰۰٪ (شکست‌ناپذیر با الگوریتم Minimax) قابل تنظیم است.

---

## ✨ ویژگی‌ها

- 🧠 **هوش مصنوعی هوشمند:** استفاده از الگوریتم Minimax برای انتخاب بهترین حرکت در سطح دشواری بالا.
- 🎚️ **تنظیم سطح دشواری:** یک اسلایدر از ۰٪ تا ۱۰۰٪ که درصد استفاده از حرکت هوشمند را مشخص می‌کند.
- 📊 **نمایش امتیازات:** شمارش بردهای انسان و کامپیوتر در طول بازی.
- 🔄 **شروع مجدد:** امکان ریست کردن صفحه بازی بدون از دست دادن امتیازات.
- 🏠 **بازنشانی کامل:** دکمه منوی اصلی برای صفر کردن امتیازات و شروع دوباره.
- 🎨 **رابط کاربری زیبا:** طراحی مدرن و رنگ‌بندی جذاب با فونت‌های واضح.
- 📢 **اعلان نوبت:** نمایش نوبت بازیکن (انسان یا کامپیوتر) با رنگ‌های مختلف.

---
